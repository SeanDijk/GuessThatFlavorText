Guess That Flavor Text

Changelog

1.2.0
  • Added the Settings page, where you can select which set(s) a random card should be chosen from
  • Fixed an issue where if the flavor text had for instance 'SPINDA' in full capitals, it would not be replaced by 'BLANK'
  • Added changelog page

1.1.0
  • Better name detection to replace the Pokémon name with "BLANK". For instance if the card is called "Galarian Moltres" and there is "Moltres" in the flavor text, it gets replaced.
  • Answers do not need to include the gender signs (♀ and ♂). So for example "Nidoran" would suffice for either "Nidoran ♀" or "Nidoran ♂"
  • Added changelog page
